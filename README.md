## Глава 2. Расширенные интерактивные возможности

-Выполнение внешнего кода **%run exapmple.py**
-Измерение времени выполнения кода **%timeit**
-Измерение времени выполнения блока кода **%%timeit**
-Документация магических методов **%magic**
-Простой список магических методов **lsmagic**

    In[...] --> list
    Out[...] --> dict 

    print(_) - доступ к предыдущим выводам _ - прошлый вывод __ - позапрошлый вывод ___ - максимальный вывод назад

    Быстрый доступ к Out[X] выглядит как _X 

**;** - Подавление вывода(результат не сохраняется в истории команд). Ставится в конце строки

**%history** - доступ к нескольких выводам

    example: %history -n 1-3 
    
**%rerun** - выполняют повторно заданную часть истории команд

**%save** - сохраняет какую-либо часть истории команд

Инструкции командного процессора в оболочке IPython

Для инструкции используется "**!**" перед командой

    example:
        In[...]: !ls
        In[...]: !pwd

Передача значений в командный процессор и из него

    example:
        In [...]: contents = !ls
        In [...]: print(contents)
        ['myproject.txt']
    example:
        In [...]: message = "Hello from Python"
        Для передачи из python используются фигурные скобки "{...}"
        In [...]: !echo {message}
            Hello from Python

Автоматические функции

    %cat, %cp, %env, %cd и т.д.
    
Их можно настраивать с помощью "магической" функции **%automagic**
